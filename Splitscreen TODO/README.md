## Controls

### Player 1 

Keyboard:
* W - Move forwards
* A - Turn left
* S - Turn Right
* D - Move backwards

### Player 2

Keyboard:
* UP ARROW - Move forwards
* LEFT ARROW - Turn left
* RIGHT ARROW - Turn Right
* DOWN ARROW - Move backwards
